# Reference Speakers

Create one folder per Egyptian Arabic speaker and place clean WAV references inside it.

Example:

- `data/reference_speakers/cairo_female_01/ref1.wav`
- `data/reference_speakers/cairo_female_01/ref2.wav`
- `data/reference_speakers/cairo_male_01/ref1.wav`

Guidelines:

- Prefer Egyptian Arabic reference speech, not MSA-only reading.
- Avoid music, overlap, phone compression artifacts, and long silences.
- Keep each clip short and clean.
- Make sure you have the speaker's consent.
